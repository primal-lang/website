Versions

Releases

The latest release of Primal is version 0.0.1. You can find all the releases on the GitHub page.

Changelog

0.0.1

Language

  • Added lexical analyzer
  • Added syntactic analyzer
  • Added semantic analyzer
  • Added support for:
    • Boolean
    • Numbers
    • Strings
  • Added main function entry point

Standard library

  • Added basic functions for:
    • Control
    • Error
    • Debugging
    • Comparison
    • Arithmetic
    • Logic
    • String
    • Casting

Runtime

  • Added runtime interpreter
  • Added lazy evaluation

Roadmap

The full list of upcoming features can be found on the GitHub project. Don't hesitate to upvote them and submit your own feature requests.

Contribute

You can help the Primal project in many ways:

  • Give us feedback using the form on the Primal website
  • Create a GitHub issue to report a bug, request a feature, or ask a question
  • Submit a GitHub PR to improve the code, documentation, tests, samples, etc.
  • Contribute to the Primal website from its own GitHub repo
  • Ask and answer Primal questions on StackOverflow
