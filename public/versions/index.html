Versions

Releases

The latest release of Primal is version 0.0.1. The full list of releases is located on GitHub. The complete changelog is available on GitHub as well.

Roadmap

The full list of upcoming features can be found on GitHub. Don't hesitate to upvote them and submit your own feature requests.

Contribute

You can help the Primal project in many ways:

  • Give us feedback using the web form
  • Create a GitHub issue to report a bug, request a feature, or ask a question
  • Submit a GitHub PR to improve the code, documentation, tests, samples, etc.
  • Contribute to the website from its own GitHub repo
  • Ask and answer Primal questions on StackOverflow
