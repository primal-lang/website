Reference

Primal has a rich set of core functions that provide essential support for developers. These functions are designed to be simple and easy to use, while also being powerful and flexible. They can be used to build complex applications and solve a wide range of problems.

They are divided in the following categories:

  • Operators
  • Control
  • Error
  • Debugging
  • Comparison
  • Arithmetic
  • Logic
  • String
  • Casting

Note: Primal doesn't support static types yet, but in order to improve the readability of the documentation, each function's signature has the parameters and the return value typed (or marked as generic when it applies).

Control

If-else
Input One conditional argument (a) and two arguments of any type (b and c). The conditional argument must evaluate to a boolean. The other two arguments must evaluate to the same type.
Output If the condition is true, it returns the first argument. Otherwise it returns the second argument.
Try-catch
Input Two arguments, which must evaluate to the same type.
Output It returns the first argument unless it evaluates to an error, in which case, it returns the second argument.

Error

Throw error
Input An argument of any type.
Output It returns an error wrapping that argument.

Debugging

Debug
Input An argument of any type.
Output It writes that argument in the standard output and returns it.
