Reference

Primal has a rich set of core functions that provide essential support for developers. These functions are designed to be simple and easy to use, while also being powerful and flexible. They can be used to build complex applications and solve a wide range of problems.

They are divided in the following categories:

Operators → Control → Error → Comparison → Arithmetic → Logic → String → List → Map → Set → Stack → Queue → Vector → Timestamp → File → Directory → Casting → JSON → Hashing → Console → Environment →

Note: Primal doesn't support static types yet, but in order to improve the readability of the documentation, each function's signature has the parameters and the return value typed.
